# Gamified Learning Platform  
*A full-stack web application that turns learning into a personalized, game-like experience.*

---

## Overview  
Gamified Learning Platform is an interactive application that helps users learn by generating personalized quiz questions as they progress. The platform adapts to users’ inputs and scores, making learning more fun, structured, and engaging.

Users can answer questions, earn points, and track progress — creating a motivating learning loop that blends education with gameplay.

I built this project to explore **full-stack development**, **LLM-powered question generation**, and UI/UX design for better learning experiences.

---

## Inspiration  
I built this project because I wanted hands-on experience integrating Generative AI into a full-stack application.
Until this point, most of my AI experience had been model-focused, so this was a way to learn how LLM-based features interact with real users, APIs, databases, and UI components.
A quiz-style learning platform felt like a practical way to experiment with prompt design, user state, and dynamic content generation while improving my understanding of end-to-end development.  

This project helped me explore how **Generative AI + web apps** can work together to create smarter self-learning tools.

---

## Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Machine Intelligence:** LLM-powered quiz generation using OpenAI API
- **Database / Auth:** Firebase / MongoDB  
- **Version Control & Deployment:** GitHub 

---

## Features
Personalized quiz generation using LLMs  
Real-time score calculation  
Progress tracking  
Gamified UI (badges, rewards, or streaks)  
Secure authentication + data persistence  

---

## What I Learned
- Building end-to-end apps using **React + Node + Firebase**  
- How to structure API calls to an LLM for contextual question generation  
- Designing a clean UI for interactive learning  
- User authentication + state management  
- Managing real-time user progress in the frontend + backend  
- Thinking about scalability and session management
